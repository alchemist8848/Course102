The C Character set

The C Character Set refers to the set of valid characters that can be used in a C program. It includes the letters of the alphabet (both uppercase and lowercase), digits, punctuation marks, and a few special characters such as the space character and the null character. The C Character Set is specified in the C standard and is used as the basis for the syntax and semantics of the C language.
Type of C character set
  • Alphabets
  • Digit
  • Special Character
  • White space
Alphabets
In C programming, small letters and capital letters are distinct. Uppercase and lowercase letters of the English alphabet (A-Z, a-z)
Digits
C programming support 10 digit (0-9) which are used to construct numerical value.
Special character
C programming supports set of special symbols that include symbols to perform mathematical operation, check condition and other special symbol. Special character %,&,*,etc.
White space
In computer programming, white space is a character that represent vertical space in typography. Enter key(\n), tab space key(\t), space key ( −), back space key (‘\b’) and so on.

Keywords are also known as reserved words and we cannot alter the meaning of keywords. Keywords are basic building block for developing C programs. There are only 32 keywords in standard C and these keywords must be written in lowercase.

Identifier

In the C programming language, an identifier is a name given to a variable, function, or any other user-defined item. An identifier is used to identify these items within the program and to reference them elsewhere in the code.
In C, identifiers can contain letters, digits, and underscores, but they must begin with a letter or an underscore. They are case-sensitive, which means that "myVariable" and "myvariable" are considered to be different identifiers.

Rules for creating a valid identifier

  • Identifiers must begin with a letter or an underscore (_).
  • Identifiers can contain letters, digits, and underscores.
  • Identifiers cannot contain any other special characters or symbols, except for the underscore.
  • Identifiers are case-sensitive, which means that "myVariable" and "myvariable" are considered to be different identifiers.
  • Identifiers cannot be a keyword of C language.
  • Identifiers should not be too long, as the compiler may not recognize them. The maximum length of an identifier is implementation-defined, but it is typically around 31 characters.
  • Identifiers should be meaningful and should represent the purpose of the variable, function or any other user-defined item it represents.

Example of Valid Identifier

  • counter
  • _private_counter
  • is_done
  • x
  • y
  • z
  • result
  • average_score
  • student_name
  • _temp

Example of InValid Identifier

  • a rea (space can not be used)
  • 1test (must start from letter)
  • my-count (hyphen is not allowed)
  • f(x) (parantheses are not allowed)
  • for (keyword can not be used as an identifier)

Variables

Variables are named locations in computer memory where values can be stored and manipulated in a program. They are used to represent values that may change during the execution of a program. Variables must be declared with a specific data type, such as integer, float, or char, which determines the type of values that can be stored in the variable. The value stored in a variable can be retrieved, modified, or used as part of a calculation in the program.

To declare a variable, specify the type and assign it a value:


                            data_type var;                          
                            
                            data_type var1, var2, var3, ...., varN;
                            
                            data_type variable_name;
                            
                            data_type comma_separated_list_of variables;

Examples


                        int var1, var2, count;
                        int x = 10;
                        float y = 3.14;
                        char z = 'a';
                        double area, num1, factorial;
                        double a = 2.71828;
                        short b = 32767;
                        long c = 123456789;
                        long int x, y, z;
                        unsigned int d = 4294967295;

Constants

In the C programming language, a constant is a value that cannot be changed during the execution of a program. Constants can be of various data types, such as integers, floating-point numbers, characters, and strings. Constants are declared using the const keyword and can be used in place of variables to provide fixed values.

To declare a constant, add const keyword, specify the datatype and assign it a value:

const datatype identifier = value; 
#define identifier value

Symbolic Constant

A symbolic constant is a name that substitutes for a sequence of characters. The characters may represent a numeric constant, a character constant or a string constant. Thus, a symbolic constant allows a name to appear in place of a numeric constant, a character constant or a string. When a program is compiled, each occurrence of a symbolic constant is replaced by its corresponding character sequence. Symbolic constants are usually defined at the beginning of a program. The symbolic constants may then appear later in the program in place of the numeric constants, character constants, etc. that the symbolic constants represent
#define   symbolic_constant_name   value_of_the_constant
