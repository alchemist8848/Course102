C-Operators

Chapter 3

An operator in a programming language is a symbol that tells the compiler or interpreter to perform specific mathematical, relational or logical operation and produce final result An Operator might have a single or multiple operands.

Arithmetic Operators

An arithmetic operator is a mathematical function that takes numeric inputs (Operands) and performs the calculation on them. The result of the calculation is then returned as the output. The calculations that can be performed using arithmetic operators include addition, subtraction, multiplication, division, modulus (remainder), and exponentiation.
Operator Meaning of Operator
+ addition or unary plus
- subtraction or unary minus
* multiplication
/ division
% remainder after division (modulo division)

Assignment Operators

Assignment operators are used to assigning value to a variable. The left side operand of the assignment operator is a variable and right side operand of the assignment operator is a value. The value on the right side must be of the same data-type of the variable on the left side otherwise the compiler will raise an error.

Relational or Comparison Operators

In C programming, relational operators such as > (greater than), (less than), >= (greater than or equal to), (less than or equal to), == (equal to), and != (not equal to) are used to compare the values of two operands and return a boolean value of either 0 or 1. These operators are used in conditional statements and loops to control the flow of a program based on the evaluation of expressions.
Operator Meaning of Operator Example
== Equal to 5 == 3 is evaluated to 0
> Greater than 5 > 3 is evaluated to 1
< Less than 5 < 3 is evaluated to 0
!= Not equal to 5 != 3 is evaluated to 1
>= Greater than or equal to 5 >= 3 is evaluated to 1
<= Less than or equal to 5 <= 3 is evaluated to 0

Logical Operators

In C programming, logical operators such as && (AND), || (OR), and ! (NOT) are used to perform operations on boolean values (i.e., 0 or 1) and return a boolean result. These operators are commonly used in conditional statements to control the flow of a program based on the evaluation of expressions.
Operator Meaning Example
&& Logical AND. True only if all operands are true If c = 5 and d = 2 then, expression ((c==5) && (d>5)) equals to 0.
|| Logical OR. True only if either one operand is true If c = 5 and d = 2 then, expression ((c==5) || (d>5)) equals to 1.
! Logical NOT. True only if the operand is 0 If c = 5 then, expression !(c==5) equals to 0.

Unary Operator

A unary operator is an operator used to operate on a single operand to return a new value. In other words, it is an operator that updates the value of an operand or expression's value by using the appropriate unary operators.
Operator Description
++ (increment operator) Increments the value of an operand by 1
-- (decrement operator) Decrements the value of an operand by 1
- (unary minus operator) Negates the value of an operand

Bitwise Operators

Operators that perform bit-level operations on integer values. They treat their operands as sequences of binary digits (bits) and operate on each bit individually.They are often used in low-level programming, such as working with hardware or encoding data.
Operators Meaning of operators
& Bitwise AND
| Bitwise OR
^ Bitwise exclusive OR
~ Bitwise complement
<< Shift left
>> Shift right

Conditional Operators

The conditional operator is also known as a ternary operator. The conditional statements are the decision-making statements which depends upon the output of the expression. It is represented by two symbols, i.e., '?' and ':'. As conditional operator works on three operands, so it is also known as the ternary operator. 

					Expression1? expression2: expression3;
